# ระบบกิจกรรมรับน้อง — ค้นหาบ้าน & เช็คชื่อ

เว็บกิจกรรมรับน้อง คณะวิทยาศาสตร์ · มี 2 บทบาทในเว็บเดียว

🔗 **เปิดเว็บ:** https://rnregis.koravit.workers.dev

| บทบาท | ใครใช้ | เข้าอย่างไร |
|---|---|---|
| 🔍 ค้นหาบ้าน | นักศึกษาทั่วไป | เปิดหน้าแรก |
| ✅ หน่วยลงทะเบียน | เจ้าหน้าที่ | เปิด `/staff` แล้วใส่รหัสผ่าน |

---

## 🔍 สำหรับนักศึกษา — ค้นหาบ้านของฉัน

1. เปิดเว็บ → กด **ยอมรับ PDPA**
2. พิมพ์ **รหัสนักศึกษา** หรือ **ชื่อ-นามสกุล** ในช่องค้นหา
3. แตะที่ชื่อตัวเอง → เห็น **เลขบ้าน** ของน้อง
4. กดปุ่ม **เข้ากลุ่มไลน์** หรือสแกน **QR** ในการ์ด เพื่อเข้ากลุ่มประจำบ้าน

> มีเช็คลิสต์ 3 ขั้นตอนคอยนำทาง และการ์ด **สายรหัส** ไว้ฝากช่องทางติดต่อให้รุ่นพี่

---

## ✅ สำหรับเจ้าหน้าที่ — เช็คชื่อ

1. เปิด `/staff` → ใส่ **รหัสผ่านเจ้าหน้าที่**
2. เลือก **ช่วงที่กำลังเช็ค** (20 มิ.ย. / 21 มิ.ย. / ออกงาน)
3. **สแกนบาร์โค้ด** หรือพิมพ์รหัสนักศึกษาแล้วกด **Enter** → เช็คชื่อทันที
4. แก้รายคนในตารางได้ · กรองตามบ้าน/สถานะได้ · กด **Export CSV** เพื่อดาวน์โหลดผล
5. ครบ **3 ช่วง** = ได้ชั่วโมงกิจกรรม

> ทุกเครื่องเห็นข้อมูลตรงกัน **แบบเรียลไทม์** — เช็คจากเครื่องไหนก็อัปเดตให้ทุกเครื่องทันที

---

ข้อมูลนักศึกษาเก็บเป็นความลับตามหลัก PDPA ใช้เฉพาะกิจกรรมรับน้องเท่านั้น
